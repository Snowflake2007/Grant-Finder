Before vs After

Previous (reference-inspired)

✕ Fixed 100vh split — no scrolling possible
✕ No navbar — no navigation or branding bar
✕ Left/right split wasted half the screen on an image
✕ Modal slid from center — no entry animation
✕ Step bar connectors were static (no done state)
✕ No "How It Works" section — user guessed the flow
✕ Purple accent colour unrelated to IBM / gov branding
✕ Form fields stacked in one long column

Now (GrantFinder original)

✓ Full scrollable page — hero + features + footer
✓ Glassmorphism sticky navbar with links + CTA button
✓ Full-width centred hero with animated gradient orbs + dot grid
✓ Modal slides up from bottom with spring animation
✓ Step bar connectors turn green when done
✓ 6-card "How It Works" feature strip, one per AI agent
✓ Blue accent (#3b82f6) matches IBM / government aesthetic
✓ Guided form uses 2-column grid — compact and scannable

All Design Improvements

Navbar Glassmorphism sticky bar
Frosted-glass effect with blur, nav links, brand icon and gradient CTA. Hides links on mobile.
Hero Animated orb background
3 slowly drifting radial-gradient orbs + subtle dot-grid overlay create depth without an image.
Stats Pill-shaped stat row
4 rounded pill stats (6 agents, 50+ schemes, <60s, India) instead of block boxes.
Features 6-card "How It Works"
Each of the 6 AI agents gets its own colour-coded card with step badge, icon, and description.
Modal Bottom-sheet slide-up
Modal slides up from the bottom edge with a spring cubic-bezier animation. Sticky header inside.
Tabs Underline tab navigation
Clean underline tabs replace the boxy button style — more modern and less cluttered in the modal.
Steps Animated progress connectors
Step bar connectors turn green (#10b981) when a step is done, giving real-time visual feedback.
Cards Hover lift effects
Feature cards, discovery cards and scheme rows all lift on hover with border-color transition.
Scrollbar Custom thin scrollbar
Slim 6px scrollbar styled to match the dark theme — no jarring system-default grey bar.
Score Large score display
Review score shown as a 36px bold number — immediately readable, no ring SVG complexity.
