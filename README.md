# 🚀 AI Grant & Funding Finder for Startups

An Agentic AI-powered platform that helps startups and entrepreneurs discover suitable funding opportunities, government grants, and startup schemes based on their profile, domain, location, startup stage, and funding requirements.

The system uses multiple specialized AI agents built with **IBM watsonx.ai** to analyze user information, evaluate eligibility, and provide personalized funding recommendations.

## ✨ Features

- 🤖 **Multi-Agent AI System**
  - Profile Agent: Extracts startup details and requirements.
  - Eligibility Agent: Matches startups with suitable funding schemes.
  - Discovery Agent: Provides grant details, documents, deadlines, and sources.

- 💡 **Personalized Recommendations**
  - Finds funding opportunities based on startup-specific information.

- 🖥️ **Interactive Frontend**
  - Built using **IBM Bob** for user-friendly interaction.

- ⚙️ **AI Backend**
  - Flask-based API integrated with IBM watsonx.ai foundation models.

## 🏗️ Tech Stack

- IBM watsonx.ai
- IBM Bob
- Python
- Flask
- IBM watsonx.ai SDK
- LLM-based Multi-Agent Architecture

## 🔄 Workflow

User Input → IBM Bob Frontend → Flask API → AI Agents → watsonx.ai Model → Funding Recommendations

## 🎯 Problem Solved

Startups often struggle to find relevant funding opportunities due to scattered information and complex eligibility requirements. This platform simplifies the process by using AI agents to identify and recommend suitable funding schemes.
